My First Webpage

My first webpage is what the name describes. This is my first webpage I have built during my time at Epicodus. Within the repository lies favorite-things. Favorite-things is a web page I built to describe my favorite things and people using HTML and CSS. People including my parents, friends, and siblings.

Skills I have learned from this project include:

  • Using ordered and unordered lists.
  • Displaying images.
  • Changing styling with inline html and using header tags.
  • how to style tags with CSS.

Cup Cake Webpage

My cupcake webpage is a display of my group’s favorite cupcakes from our imaginary shop using HTML and CSS. A simple website with pictures of our favorite cupcakes.

Skills I have learned from this project include:

  • Usage of classes
  • Usage of IDs
  • Usage of span tags
  • Greater understanding of HTML and CSS

Evie's Webpage

Evie’s webpage is the first web page I created using HTML without any hand holding from Epicodus. The web page includes details about my dog Evie including her favorite treats, activities, and a tiny bio about her.

Skills I have learned from this project include:

  • An understanding of HTML and CSS.
  • Learning how block elements work in HTML
  • Displaying images
  • Using ordered and unordered lists
